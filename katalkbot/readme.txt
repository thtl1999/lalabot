[라라봇 도움말]

[원본 짤 검색] - 픽시브에서 원본 짤을 찾습니다.
'라라봇' -> 사진 올리기
'라라봇설정60' -> 정확도가 60% 이하면 세부정보를 감춥니다

[짤 검색] - danbooru에서 짤을 찾습니다.
'라라찾아 랜덤' -> 랜덤한 짤을 보여줍니다
'라라찾아 manaka lala' -> manaka_lala 의 태그를 가진 짤을 랜덤으로 보여줍니다
캐릭터, 특징, 작가등 여러가지를 검색할 수 있습니다.
한국어가 있을 경우 일본어 로마자로 변환합니다.
예)라라찾아 후지와라 치카 -> '라라찾아 fujiwara_chika'
하지만 안되는 경우도 있습니다
예)마나카 라라 -> 'manaka_rara' (마나카 라라는 'manaka_lala' 이므로 검색 안됨)

이미지를 불러올 수 없는 작품은 썸네일만 표시되며 이미지를 클릭하면 원본 링크로 연결됩니다.

[단어 익히기]
'라라 배워' -> 'A=B'
톡방에서 A가 입력되면 라라가 B를 말합니다
'라라 잊어' -> 'A'
A에 할당된 단어를 잊습니다

[리겜 곡 추천]
'라라사볼20'
'라라디디알15'
'라라팝픈46'
선택한 난이도의 곡 중에서 랜덤으로 추천해줍니다

'라라사볼.17.18.19'
한번에 여러 곡을 추천해줍니다

'라라사볼20리스트'
선택한 난이도의 곡을 모두 보여줍니다

'라라사볼18ㅇ'
사볼 18렙 곡을 하나 추천해주고 제목으로 유튜브 검색을 합니다

사볼곡 데이터: sdvx.in
팝픈곡 데이터: https://hellwork.jp/popn/wiki/
디디알곡 데이터: https://www21.atwiki.jp/asigami

[율전동 밥추천]
'라라혼밥' -> 혼자서 먹을 수 있는 곳을 추천해줍니다
'라라밥' -> 혼밥집을 포함해서 모든 먹을곳을 추천해줍니다
'라라회식' -> 비싼곳을 추천해줍니다
'라라밥 리스트' -> 등록된 모든 식당을 보여줍니다

[유튜브 검색]
'라라유튜브 검색할것' -> 유튜브를 검색해서 가장 처음에 나온 영상을 보여줍니다
'다음' -> 검색결과의 다음 영상들을 보여줍니다

[미세먼지 정보]
'미세먼지' -> 율전동 미세먼지 정보를 보여줍니다
정보: http://kweather.co.kr/air/air_area_realtime_popup6.html?acode=4111113300
